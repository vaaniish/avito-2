# Актуальный аудит

В этой папке хранится только актуальная audit-документация по текущему состоянию проекта.

## Файлы

- `current-structure-audit.md` — текущее состояние структуры, сильные стороны, риски и наблюдения
- `current-hotspots.md` — самые крупные и потенциально рискованные файлы после рефакторинга
- `current-verification-report.md` — последние подтверждённые проверки сборки и тестов

## Принцип

Исторические phase-логи, промежуточные baseline-отчёты и устаревшие decision-логи здесь не хранятся. Если документ больше не описывает текущее состояние репозитория, он не должен оставаться в `docs/audit`.
